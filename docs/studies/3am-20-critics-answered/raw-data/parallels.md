- Rev 14:7 → Isa 34:10 (ot, vocabulary — 'smoke ascending forever' formula)
- Rev 14:7 → Acts 4:24 (nt, vocabulary + structural — creation formula with poieo)
- Rev 14:7 → Rev 15:4 (nt, vocabulary + theological — fear, glorify, worship parallels)
- Rev 14:7 → Acts 14:15 (nt, vocabulary + structural — creation formula with poieo)
- Rev 14:11 → Isa 34:10 (ot, vocabulary + structural — identical smoke-forever formula for temporal destruction)
- Rev 17:1-5 → 1 Kgs 14:24 (ot, vocabulary — abomination and prostitution/harlotry)
- Rev 14:7 → Rev 19:1 (nt, vocabulary — glory, heaven, loud voice)
