# Cross-Testament Parallels

- Rev 14:7 → Exo 20:11 (ot, verbal + thematic + structural — Four shared terms in sequence (poieo/asah, ouranon/shamayim, gen/erets, thalassan/yam). Same verb root (poieo = LXX rendering of asah). The fourth element diverges (pegas hydaton vs. panta ta en autois). Strongest OT allusion in Revelation per Paulien.)
- Rev 14:7 → Acts 4:24 (nt, verbal — Both use the creation formula with poieo + ouranon + gen + thalassan. Acts 4:24 follows Exo 20:11 exactly with 'panta ta en autois.' Rev 14:7 departs with 'pegas hydaton.')
- Rev 14:7 → Acts 14:15 (nt, verbal — Same creation formula as Acts 4:24. Also uses 'panta ta en autois.' Confirms the standard NT creation formula.)
- Rev 14:7 → Psa 146:6 (ot, verbal — Same creation formula. Uses 'panta ta en autois' in the LXX. Rev 14:7's departure is conspicuous against this consistent pattern.)
- Rev 14:7 → Neh 9:6 (ot, verbal + thematic — Most expansive version of the creation formula. Uses asah + shamayim + erets + yam + all therein. Connects creation to worship explicitly.)
- Rev 14:7 → Acts 17:24-31 (nt, thematic + structural — Creator God (poieo, v.24) + universal scope (v.26,30) + judgment day (v.31). Closest NT structural parallel to Rev 14:7's combination of creation, worship, and judgment.)
- Rev 14:7 → Ecc 12:13-14 (ot, structural — Fear God + commandments + judgment = the first angel's message template. The Ecclesiastes triad recurs in four OT farewell addresses.)
- Rev 14:7 → Rev 10:6 (nt, vocabulary contrast — Both describe God as Creator of heaven, earth, sea. Rev 10:6 uses ktizo; Rev 14:7 uses poieo. The verb switch within the same book is deliberate.)
- Rev 14:7 → Gen 7:11 (ot, vocabulary — 'Fountains of the great deep' (ma'yanoth, Hebrew cognate of pege). Rev 14:7's 'pegas hydaton' may layer a Flood judgment allusion onto the creation allusion.)
- Rev 14:12 → Rev 12:17 (nt, structural inclusio — Both use tereo + entolas + tou theou. Rev 12:17 opens the cosmic conflict section; Rev 14:12 closes the three angels' messages. The creation formula of 14:7 sits within this commandment frame.)
- Rev 14:12 → 1 Cor 7:19 (nt, vocabulary — Same tereo/entole/theos triad. Paul separates circumcision (ceremonial) from commandments of God (moral).)
- Heb 4:9 → Gen 2:2-3 (ot, thematic — Heb 4:4 quotes Gen 2:2. The sabbatismos remaining for God's people (4:9) is patterned on God's seventh-day rest at creation.)
- Heb 4:9 → Psa 95:7-11 (ot, quotation — Heb 3:7-11 quotes Psa 95 extensively. The 'rest' that the wilderness generation failed to enter is the foundation for the sabbatismos argument.)
- Exo 31:17 → Rev 7:3 (nt, thematic — Sabbath as owth (sign, Exo 31:17) parallels seal of God (sphragis, Rev 7:3). If the Sabbath is the sign/seal, the mark of the beast (Rev 13:16) is its counterfeit.)
- Gen 2:2-3 → Isa 66:22-23 (ot, structural bookend — Sabbath at creation (Gen 2) and in the new creation (Isa 66:22-23). The bookend suggests permanence, though Isa 66:23-24 complications (New Moon, dead bodies) require honest engagement.)
- Rev 14:7 → Rev 4:11 (nt, vocabulary contrast — Rev 4:11 uses ktizo for creation praise; Rev 14:7 uses poieo. Both are creation-worship texts in Revelation, but the verb choice distinguishes them. Rev 14:7 is specific (Exo 20:11 allusion); Rev 4:11 is generic.)
- Luke 23:56 → Rev 14:12 (nt, vocabulary — Luke 23:56 applies entole (G1785) to the Sabbath commandment ('kata ten entolen'). Rev 14:12 uses entole for 'commandments of God.' The same word covers Sabbath-keeping and the eschatological remnant's commandment-keeping.)
- Rev 14:7 → Rev 15:3-4 (nt, thematic — Vindication worship echoing the first angel: fear + glorify + worship + judgment. The same four elements recur in the post-judgment song.)
- Col 2:16 → 1 Chr 23:31 (ot, structural — The heorte/neomenia/sabbaton triad in Col 2:16 matches the annual/monthly/weekly ceremonial calendar triad in 1 Chr 23:31 and 2 Chr 31:3, identifying ceremonial rather than weekly sabbaths.)
- Rev 14:7 → Rom 1:25 (nt, thematic inversion — Rom 1:25 describes worshipping creature over Creator (using ktizo). Rev 14:7 commands worshipping the Creator (using poieo). The first angel reverses the Romans 1 chain.)
