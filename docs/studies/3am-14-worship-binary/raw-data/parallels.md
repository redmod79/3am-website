# Cross-Testament Parallels

- Rev 13:4 → Dan 7:21 (ot, vocabulary (war, beast))
- Rev 13:4 → Rev 13:2 (nt, vocabulary + structural (beast, authority, dragon))
- Rev 13:4 → Rev 13:12 (nt, vocabulary (worship, beast, authority))
- Rev 14:7 → Gen 28:17 (ot, vocabulary (fear, heaven))
- Rev 14:7 → Jonah 1:9 (ot, vocabulary (fear, heaven, sea))
- Rev 14:7 → Acts 4:24 (nt, vocabulary (heaven, made, sea, voice))
- Rev 14:7 → Acts 14:15 (nt, vocabulary (heaven, made, sea) — creation language)
- Rev 14:7 → Rev 15:4 (nt, vocabulary (worship) + theological)
- Rev 14:7 → Rev 10:6 (nt, vocabulary (heaven, sea) — creation language with ktizo instead of poieo)
- Rev 13:15 → Dan 3:6 (ot, structural (image + worship + death decree))
- Rev 13:15 → 2 Chr 23:17 (ot, vocabulary (image, kill))
