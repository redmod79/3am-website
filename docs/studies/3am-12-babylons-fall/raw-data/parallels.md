# Cross-Testament Parallels

- Rev 14:8 → Isa 21:9 (ot, verbatim quotation — doubled fall formula (naphlah naphlah babel → epesen epesen Babylon))
- Rev 14:8 → Rev 18:2 (nt, internal repetition — same doubled formula with desolation expansion)
- Rev 14:8 → Rev 16:19 (nt, polarity flip — terminal genitive swaps from porneia to orge)
- Rev 18:2 → Isa 13:21-22 (ot, vocabulary — desolation/demon-dwelling imagery (wild beasts, owls, satyrs))
- Rev 18:2 → Isa 34:13-14 (ot, vocabulary — habitation of dragons, satyrs, screech owls)
- Rev 18:2 → Jer 50:39 (ot, vocabulary — wild beasts dwell, never inhabited)
- Rev 18:4 → Jer 51:6 (ot, verbal parallel — 'flee out of Babylon' → 'come out of her')
- Rev 18:4 → Jer 51:45 (ot, verbatim — 'My people, go ye out' → 'Come out of her, my people')
- Rev 18:4 → Isa 48:20 (ot, verbal parallel — 'Go ye forth of Babylon')
- Rev 18:5 → Jer 51:9 (ot, verbal parallel — sins/judgment reaching heaven)
- Rev 18:6 → Jer 50:29 (ot, verbal parallel — recompense according to deeds)
- Rev 18:7 → Isa 47:7-8 (ot, verbatim quotation — queen hubris soliloquy ('I sit as queen, am no widow'))
- Rev 18:7 → Zeph 2:15 (ot, verbal parallel — 'I am, and there is none beside me' hubris formula)
- Rev 18:8 → Isa 47:9 (ot, verbal parallel — 'in one day' sudden judgment + sorceries)
- Rev 18:8 → Jer 51:58 (ot, verbal parallel — burned with fire)
- Rev 18:20 → Jer 51:48 (ot, verbal parallel — heaven rejoices over Babylon's fall)
- Rev 18:21 → Jer 51:63-64 (ot, verbal parallel + adaptation — stone into Euphrates → millstone into sea)
- Rev 18:21 → Matt 18:6 (nt, vocabulary — mylos (millstone) + thalassa (sea) in judgment context)
- Rev 18:22 → Ezek 26:13 (ot, verbal parallel — silenced musicians/harps)
- Rev 18:22-23 → Jer 25:10 (ot, verbal parallel — bridegroom/bride + millstone + lamp silenced)
- Rev 18:22-23 → Jer 7:34 (ot, verbal parallel — bridegroom/bride silenced)
- Rev 18:23 → Nah 3:4 (ot, verbal parallel — sorcery (kesheph/pharmakeia) deceiving nations)
- Rev 18:23 → Isa 23:8 (ot, vocabulary — emporoi as rulers/great men of earth)
- Rev 18:24 → Jer 51:49 (ot, verbal parallel — blood of slain found in Babylon)
- Rev 18:24 → Matt 23:34-35 (nt, thematic — blood of prophets from Abel to Zechariah → expanded bloodguilt)
- Rev 18:24 → Rev 6:10 (nt, altar vindication arc — martyrs' cry answered)
- Rev 17:1 → Jer 51:13 (ot, verbatim — 'sits on many waters')
- Rev 17:4 → Jer 51:7 (ot, verbal parallel — golden cup (kowce zahab → poterion chrysoun))
- Rev 18:12-13 → Ezek 27:12-22 (ot, structural + vocabulary — cargo list modeled on Tyre's trade inventory)
- Rev 19:3 → Isa 34:10 (ot, verbal parallel — smoke going up forever)
- Rev 16:12 → Isa 44:27-28 (ot, typological — dried Euphrates for Cyrus → dried Euphrates for kings of east)
