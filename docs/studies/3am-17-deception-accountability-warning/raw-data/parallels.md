# Cross-Testament Parallels

- 2 Thess 2:11 → Psa 81:11-12 (ot, vocabulary + theological: judicial abandonment pattern)
- 2 Thess 2:11 → Isa 6:9-10 (ot, theological: judicial hardening confirms chosen condition)
- Rom 1:24-28 → Psa 81:11-13 (ot, vocabulary + theological: paradidomi/shalach = gave them over)
- Isa 6:9-10 → Matt 13:14-15 (nt, direct quotation: LXX tradition with agency shift)
- Isa 6:9-10 → Mark 4:10-12 (nt, shorter quotation: parables as judicial function)
- Num 15:30 → Heb 10:26-29 (nt, theological: beyad ramah -> hekousios hamartanonton)
- Rev 15:2 → Exo 15:1-2 (ot, structural: victors by the sea singing deliverance songs)
- Rev 14:6 → Dan 6:26 (ot, vocabulary: every nation/tongue/people universal proclamation)
- Dan 12:1 → Rev 15:5-8 (nt, theological: pre-deliverance finalization convergence)
- Dan 12:1 → Rev 22:11 (nt, theological: book finalized + destinies declared fixed)
- Acts 17:30 → Rom 3:25 (nt, vocabulary + theological: hupereidon / paresis = overlooking sins)
- Rev 22:11 → Rom 3:5 (nt, vocabulary: righteousness/unrighteousness binary)
